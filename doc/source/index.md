# Welcome to Hexo Theme Almagest

This is the demonstration blog for **Hexo Theme Almagest**, a minimal and elegant Hexo theme inspired by astronomy and constellation themes.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Light & Dark Themes**: Automatic theme switching based on system preferences
- **Fast & Lightweight**: Optimized for performance with minimal dependencies
- **Customizable**: Easy to customize through configuration files
- **Multi-language Support**: Built-in support for multiple languages
- **Code Highlighting**: Beautiful syntax highlighting for code blocks
- **SEO Friendly**: Optimized for search engines

### Astronomy Theme

Fully responsive design that adapts to any screen size, from mobile phones to large desktop monitors.

### Dark Mode Support

Automatic dark mode detection with manual toggle option. Your preference is saved locally.
